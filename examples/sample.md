# Chapter One

The converter treats level-one Markdown headings as EPUB chapters.
Paragraphs split across several OCR lines are joined into readable blocks.

## A Smaller Heading

Centered HTML exported by some PDF-to-Markdown tools is preserved:

<div style="text-align: center;">A centered line</div>
